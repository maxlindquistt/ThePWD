# app-window

A web component that represents an application window, providing a container for app content with window-like behavior.

## Attributes

### data-app

Description of the attribute. Identifies the application associated with the window.

## Events

| Event Name      | Fired When                        |
|-----------------|-----------------------------------|
| mouseDown       | The mouse is clicked              |
| mouseUP         | The mouse button is released      |
| mouseMove       | The mouse is moved                |

## Example

```html
<app-window data-app="memory"></app-window>
```
